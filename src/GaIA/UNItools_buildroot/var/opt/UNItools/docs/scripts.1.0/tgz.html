NAME

tgz - Make a gzip'd tar archive  

PACKAGE

UNItools  

SYNOPSIS

tgz [destination [source...] ]  

DESCRIPTION

tgz makes a gzip'd tar archive $1 (or stdout) out of specified files (or, if not specified, from everything in the current directory).

Requires gzip in the user's path.

Requires gnu tar (or something close) in the user's path due to use of --exclude, --totals and -S.  

COMMANDS

sh(1), tar(1), gzip(1)
