   23 What does "setup" do on my computer?
   24 ====================================
   25 Setup will ask you for a partition on your harddisk,
   26 where you would like to have Linux installed.
   27 You can then choose to install this distribution from
   28 your local harddisk or via network (NFS).
   29 setup will install a very basic system.
   30 At the end, you can install lilo, the Linux Loader.
   31 Rebooting your computer with this little linux system
   32 and then runing the setup-program on it, will give
   33 the possibility to install more packages.
   34 
   35 Creating a Linux-partition:
   36 ===========================
   37 To install this Linux distribution, you need to
   38 create an extra partition on your harddisk for
   39 Linux.
