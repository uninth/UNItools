NAME
       itop - top-like interrupt load monitor

PACKAGE
       UNItools

SYNOPSIS
       itop [-n][-a]

BESKRIVELSE
       itop(1) is a glorified Linux centric vesion of cat /proc/interrupts.

OPTIONS
       -a     Display information on all interrupts.

       -n number
	      Stop  after  fInumber  interactions. May kill the application in
	      suc a way, that the tty becomes useless.

INTERRUPT TABLE
       Kernel dependent.

COMMANDS
       Written in C(1), but requires procfs(1)0

SE OGS.ad
       Dokumentationen for UNItools.

DIAGNOSTICS
       None.

BUGS
       Report them to the author.
